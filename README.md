# Andrés Pérez Araya — Portfolio

Personal portfolio website built with [Reflex](https://reflex.dev/) (Python) and exported as a static Next.js/React site.

🌐 **Live site:** [l1k4root.github.io](https://l1k4root.github.io)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | [Reflex](https://reflex.dev/) 0.6.7 |
| Frontend | Next.js 14 / React 18 |
| Styling | PostCSS / Tailwind |
| Hosting | GitHub Pages |

## Project Structure

```

├── extra_*.jpg            # Extra/hobby photos
└── index.html             # Static export entry point
```

## Customizing Content

All portfolio content is driven by **`data/data.json`**. Edit that file to update:

| Key | Description |
|-----|-------------|
| `name` / `skill` / `location` | Header info |
| `about` | Bio paragraph |
| `media` | Email, CV path, GitHub, LinkedIn links |
| `technologies` | Tech icons shown in the skills section |
| `experience` | Work history entries |
| `projects` | Featured projects with images and links |
| `training` | Education and certifications |
| `extras` | Hobbies / side interests |

Technology icons use class names from [Devicon](https://devicon.dev/) (e.g. `devicon-python-plain`).

## Local Development

### Prerequisites

- Python ≥ 3.8
- [Bun](https://bun.sh/) or Node.js

### Install & Run

```bash
# Install Reflex
pip install reflex

# Start the development server
reflex run
```

The site will be available at `http://localhost:3000`.

### Export Static Site

```bash
reflex export --frontend-only
```

The static files will be generated and placed at the repository root, ready for GitHub Pages deployment.
